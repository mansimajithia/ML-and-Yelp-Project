Data Cleanup and Exploration

The original dataset was made available by Yelp. This dataset was large and included various types of businesses. Due to the size of this data, we imported the data to Amazon AWS and cleaned up the data through Google Colab.


After data cleanup, we started preparing for Machine Learning by one-hot encoding each feature. We also added an additional feature, density of restaurants nearby.
